# GrassTab 移动端测试清单

## 准备工作

### 开发环境测试
1. ✅ 运行 `npm run dev`
2. ✅ 按 F12 打开 Chrome DevTools
3. ✅ 按 Ctrl+Shift+M 切换设备模拟器
4. ✅ 选择移动设备预设

### 实际设备测试
1. 运行 `dev-mobile.bat`
2. 记录显示的局域网地址
3. 在手机浏览器中访问该地址
4. （可选）添加到主屏幕
