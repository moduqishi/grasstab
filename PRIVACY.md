# GrassTab 隐私政策 / Privacy Policy

## 中文版

**最后更新日期**: 2025年12月8日

### 概述

GrassTab（以下简称"本扩展"）尊重并保护用户的隐私。本隐私政策说明了我们如何处理您的数据。

### 数据收集

**本扩展不收集任何个人数据。**

所有数据均存储在您的浏览器本地存储（localStorage）中，包括：
- 应用快捷方式配置
- Dock 栏设置
- 壁纸选择
- 系统设置
- 便签内容
- AI API 配置（如果您选择使用）

### 数据存储

- 所有数据仅存储在您的本地浏览器中
- 数据不会上传到任何服务器
- 数据不会与第三方共享
- 卸载扩展后，所有本地数据将被清除

### 权限说明

本扩展请求以下权限，仅用于以下目的：

1. **storage** - 在浏览器本地存储您的设置和数据
2. **unlimitedStorage** - 允许存储大量自定义内容（如自定义壁纸）
3. **host_permissions (https://*/\*)** - 用于以下功能：
   - 获取网站图标（favicon）
   - 搜索建议功能
   - 嵌入式网页小组件

### 外部服务

本扩展可能访问以下外部服务，但**不会**发送您的个人信息：

1. **图标服务** - 自动获取网站图标
   - icon.horse
   - logo.clearbit.com
   - 其他 favicon 服务

2. **搜索建议** - 提供搜索建议（可选功能）
   - Google Suggest
   - Bing Suggest
   - Baidu Suggest
   - DuckDuckGo

3. **AI 服务** - 如果您配置了 AI 助手
   - OpenAI
   - Anthropic (Claude)
   - Google (Gemini)
   - 其他 AI 提供商
   - **注意**: 您需要自行配置 API 密钥，我们不存储或传输您的密钥

### 用户控制

您完全控制您的数据：
- 可以随时导出配置文件备份
- 可以随时导入配置恢复数据
- 可以随时清除所有数据（恢复默认设置）
- 可以随时卸载扩展删除所有数据

### 安全性

- 所有数据仅存储在您的本地浏览器
- 不涉及服务器传输
- 开源代码，透明可审计

### 儿童隐私

本扩展不针对 13 岁以下儿童，也不会故意收集儿童的个人信息。

### 政策变更

我们可能会不时更新本隐私政策。任何更改将在此页面发布，并更新"最后更新日期"。

### 联系我们

如果您对本隐私政策有任何疑问,请通过以下方式联系我们:
- GitHub Issues: https://github.com/moduqishi/grasstab/issues
- Email: [您的邮箱]

## English Version

**Last Updated**: December 8, 2025

### Overview

GrassTab ("the Extension") respects and protects user privacy. This privacy policy explains how we handle your data.

### Data Collection

**The Extension does not collect any personal data.**

All data is stored in your browser's local storage (localStorage), including:
- App shortcuts configuration
- Dock bar settings
- Wallpaper selection
- System settings
- Notes content
- AI API configuration (if you choose to use it)

### Data Storage

- All data is stored locally in your browser only
- Data is not uploaded to any servers
- Data is not shared with third parties
- All local data will be cleared when the extension is uninstalled

### Permission Usage

The Extension requests the following permissions for these purposes only:

1. **storage** - Store your settings and data in browser local storage
2. **unlimitedStorage** - Allow storing large amounts of custom content (e.g., custom wallpapers)
3. **host_permissions (https://*/\*)** - Used for:
   - Fetching website favicons
   - Search suggestion functionality
   - Embedded webpage widgets

### External Services

The Extension may access the following external services, but **does not** send your personal information:

1. **Icon Services** - Auto-fetch website icons
   - icon.horse
   - logo.clearbit.com
   - Other favicon services

2. **Search Suggestions** - Provide search suggestions (optional feature)
   - Google Suggest
   - Bing Suggest
   - Baidu Suggest
   - DuckDuckGo

3. **AI Services** - If you configure AI assistant
   - OpenAI
   - Anthropic (Claude)
   - Google (Gemini)
   - Other AI providers
   - **Note**: You need to configure your own API keys, we do not store or transmit your keys

### User Control

You have complete control over your data:
- Export configuration file for backup anytime
- Import configuration to restore data anytime
- Clear all data anytime (reset to defaults)
- Uninstall extension to delete all data anytime

### Security

- All data is stored locally in your browser only
- No server transmission involved
- Open source code, transparent and auditable

### Children's Privacy

The Extension is not directed to children under 13 and does not knowingly collect personal information from children.

### Policy Changes

We may update this privacy policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date.
