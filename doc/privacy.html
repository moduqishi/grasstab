ğŸ”‘ Permission Usage

The Extension requests the following permissions for these purposes only:

1. storage

Store your settings and data in browser local storage, including app layout, wallpapers, notes, etc.

2. unlimitedStorage

Used for the following features, without collecting or transmitting your personal information:

  • Fetch website icons - Auto-fetch favicons for added websites
  • Search suggestions - Provide search suggestions (optional)
  • Embedded webpage widgets - Allow embedding web content in widgets

Search Suggestion Services (Optional)

  • Google Suggest
  • Bing Suggest
  • Baidu Suggest
  • DuckDuckGo

Only used when you use the search bar, providing suggestions based on your input.

AI Services (Optional)

Important: AI features require you to configure your own API keys. We do not store or transmit your API keys to any servers. All API calls are sent directly from your browser to AI service providers.

Supported AI services:
