## 📝 发布步骤

### 步骤 1: 生成图标

1. 打开浏览器访问项目中的 `generate-icons.html`
2. 点击 "✨ 生成所有图标" 按钮
3. 下载三个 PNG 文件到 `chrome-extension` 文件夹:
   - icon16.png
   - icon48.png
   - icon128.png

### 步骤 2: 创建扩展包

运行打包命令:
```bash
.\build-extension.bat
```

这将:
1. 构建项目 (`npm run build`)
2. 创建 `chrome-extension` 文件夹
3. 复制所有必需文件
4. 生成图标文件

### 步骤 3: 创建 .zip 包

手动创建 ZIP 包:
1. 进入 `chrome-extension` 文件夹
2. 选中所有文件 (不要选择文件夹本身)
3. 右键 → 发送到 → 压缩(zipped)文件夹
4. 命名为 `grasstab-edge.zip`

**重要**: ZIP 包内容应该直接是文件,不要包含外层文件夹!

正确结构:
```
grasstab-edge.zip
├── manifest.json
├── index.html
├── icon16.png
├── icon48.png
├── icon128.png
└── assets/
    ├── index-xxx.css
    └── index-xxx.js
```

## 🚀 在合作伙伴中心提交

### 1. 注册开发者账号
访问: https://partner.microsoft.com/dashboard/microsoftedge/

### 2. 创建新扩展
1. 登录合作伙伴中心
2. 点击 "创建新扩展"
3. 进入扩展配置页面

### 3. 上传扩展包
1. 在左侧菜单选择 "包"
2. 上传 `grasstab-edge.zip`
3. 等待验证通过
4. 点击 "继续"

### 4. 配置可用性
**可见性**: 选择 "Public (公开)"
**市场**: 建议选择所有市场

### 5. 填写属性
- **类别**: "生产力" 或 "外观"
- **隐私策略**: 
  - 如不收集数据: 选择 "否"
  - 如收集数据: 提供隐私策略 URL
- **网站 URL** (可选): 您的项目网站
- **支持联系**: 提供支持邮箱

### 6. 添加商店一览详细信息

#### 英语 (en-US)
1. 点击英语行的 "编辑详细信息"
2. 填写:
   - **扩展名称**: GrassTab - New Tab
   - **描述**: (使用上面准备的英文描述)
   - **扩展徽标**: 上传 128x128 图标
   - **屏幕截图**: 上传 3-6 张
   - **搜索词**: new tab, productivity, dashboard, 等

#### 中文 (zh-CN)
1. 在下拉菜单添加中文语言
2. 填写:
   - **扩展名称**: GrassTab - 新标签页
   - **描述**: (使用上面准备的中文描述)
   - **扩展徽标**: 复制英语的徽标
   - **屏幕截图**: 复制英语的截图
   - **搜索词**: 新标签页, 生产力, 桌面, 效率工具, 等

## 📞 常见问题

**Q: 审核需要多久?**
A: 通常 3-7 个工作日,复杂情况可能更长

**Q: 审核未通过怎么办?**
A: 查看拒绝原因,修改后重新提交

**Q: 可以更新已发布的扩展吗?**
A: 可以,上传新版本 ZIP 包即可,会再次进入审核

**Q: 扩展名称可以修改吗?**
A: 需要更新 manifest.json 中的 name 字段并重新上传包

**Q: 不收集数据需要隐私策略吗?**
A: 如果确实不收集任何数据,可以选择 "否"
